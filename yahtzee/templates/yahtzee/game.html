Yahtzee V2 - Overview

This updated version of yahtzee is written as a Django (Python) application. This incorporates a database, allowing games to be saved and returned to at a later time. The game is served to the browser via html templates, modeled on V1 of the game, but with data taken from the database.

Players can:
- log in as a user (of the jakjarvis apps suite) or play as a Guest
- setup a game between two registered users
- return to the unique url for their game at any time

Source Code
